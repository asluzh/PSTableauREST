# Get-TableauCurrentUserId

## SYNOPSIS
Returns the current user ID

## SYNTAX

```
Get-TableauCurrentUserId
```

## DESCRIPTION
Returns the user ID of the currently signed in user (stored in an internal module variable)

## EXAMPLES

### EXAMPLE 1
```
$userId = Get-TableauCurrentUserId
```
