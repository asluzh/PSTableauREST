# Get-TableauSiteSettingsEmbedding

## SYNOPSIS
Get Embedding Settings for a Site

## SYNTAX

```
Get-TableauSiteSettingsEmbedding
```

## DESCRIPTION
Returns the current embedding settings for the current site.

## EXAMPLES

### EXAMPLE 1
```
$settings = Get-TableauSiteSettingsEmbedding
```
