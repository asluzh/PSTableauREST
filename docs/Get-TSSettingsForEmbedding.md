# Get-TSSettingsForEmbedding

## SYNOPSIS
Get Embedding Settings for a Site

## SYNTAX

```
Get-TSSettingsForEmbedding
```

## DESCRIPTION
Returns the current embedding settings for the current site.

## EXAMPLES

### EXAMPLE 1
```
$settings = Get-TSSettingsForEmbedding
```
