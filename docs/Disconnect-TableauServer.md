# Disconnect-TableauServer

## SYNOPSIS
Sign Out

## SYNTAX

```
Disconnect-TableauServer
```

## DESCRIPTION
Signs you out of the current session.
This call invalidates the authentication token that is created by a call to Connect-TableauServer.

## EXAMPLES

### EXAMPLE 1
```
Disconnect-TableauServer
```
