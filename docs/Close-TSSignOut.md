# Close-TSSignOut

## SYNOPSIS
Sign Out

## SYNTAX

```
Close-TSSignOut
```

## DESCRIPTION
Signs you out of the current session.
This call invalidates the authentication token that is created by a call to Open-TSSignIn.

## EXAMPLES

### EXAMPLE 1
```
Close-TSSignOut
```
